    Software

    Return to home page Github


    These are few of the open-source software projects I have worked on. For a more exhaustive list, check out my Github profile. For my contributions to closed-source software projects, refer to my CV.

    paperfetcher

    Python package to automate handsearching and citation chasing for systematic reviews by retrieving bulk bibliographic metadata from various databases.

    Github Website Documentation

    paperfetcher-web-app

    Streamlit web-app to automate handsearching and citation chasing for systematic reviews.

    Github Website

    WHAM

    Python package to construct free energy profiles from multiple biased simulation trajectories using log-likelihood maximization.

    Github Documentation

    pyves

    Python package to perform variationally enhanced sampling in OpenMM using PyTorch and OpenMM-torch for computing analytical and neural network bias potentials and collective variables.

    Github Documentation

    stringmethod

    Python implementation of the string method to compute the minimum energy path between two points in an energy landscape.
