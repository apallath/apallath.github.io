      Molecular simulation & analysis


    WHAM

    Python package to construct free energy profiles from multiple biased simulation trajectories using log-likelihood maximization.

    Github Documentation

    INDUSAnalysis

    Python package to generate input data for and analyze molecular dynamics trajectories generated by GROMACS patched with INDUS.

    Github Currently private Documentation Currently private

    pyves

    Python package to perform variationally enhanced sampling in OpenMM using PyTorch and OpenMM-torch for computing analytical and neural network bias potentials and collective variables.

    Github Documentation

    stringmethod

    Python implementation of the string method to compute the minimum energy path between two points in an energy landscape.

    Github

    pyib

    PyTorch & OpenMM implementations of deep learning based approaches for learning and biasing reaction coordinates in molecular simulations.

    Github

    Bootstrapping Tutorial

    Jupyter notebook tutorial on bootstrapping (and other related methods) for resampling and calculating uncertainties from correlated timeseries data.

    Github

    gcmc
